# Mockbros Frontend

Mockbros is a mock interview product. This frontend skeleton focuses on the hackathon demo flow:

1. Choose an interview type.
2. Answer interview questions.
3. Submit answers.
4. Review AI feedback.

## Palette

- Primary: `#FFFFFF`
- Secondary: `#000000`
- Accent: `#FFD700`

These are defined as CSS variables in `src/styles.css`.

## Run Locally

```bash
npm install
npm run dev
```

The Vite dev server runs on `http://localhost:5173`.

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```
